## Json file
##### botconfig.json
Make sure you make a json file like this:
```json
{
    "token": "Your Bot Token",
    "prefix": "!",
    "embedfooter": "Test Bot | Username#0000",
    "botOwner": "bot.guilds.resolve(Guild You're In).members.resolve(Your User Id)"
}
```
